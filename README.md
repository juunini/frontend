## Usage

### Install

```bash
yarn && yarn postinstall
```

### Running Dev Server

```bash
yarn dev
```

### Lint

```bash
yarn lint
```

### Unit Test

```bash
yarn test
yarn test:watch
yarn test:coverage
```

### E2E Test

```bash
yarn cypress
yarn cypress:headless
yarn e2e
yarn e2e:headless
```

### Storybook

```bash
yarn storybook
yarn build-storybook
```

### Build and Export

```bash
yarn build
yarn start
yarn export
```
